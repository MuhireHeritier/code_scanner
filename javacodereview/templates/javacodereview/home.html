Welcome to the Code Review!

The Code Review enables a registered user to upload code snippets or a code file and review all the codes for potentical vulnerabilities and generate a report on specific parts of the codes that are in need of refactoring and any other important comments.

Today, you can join a community of other programmers and developers who trusted this application to ease their development process and hence enhance fast performance and robustness.

This system is automated, so it scans every line of code with quality, speed, and accuracy. This protytpe was developed using Django- a python based web framework and open source. It provides security that handles the most common web application security flaws such as user authentication, authorization, and cross-site injections.
